## Запуск кода
Для запуска достаточно с папке с проектом выполнить 
```
python test1.py
```
или
```
python test3.py
```
При выполнении задания не используются сторонние библиотеки, поэтому ничего дополнительно устанавливать не нужно, кроме самого Python
